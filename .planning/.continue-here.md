## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
| --- | --- | --- | --- |
| Date object effect loop | `DateControl`에서 매 렌더마다 새 `Date` 객체를 effect 의존성에 넣어 `Maximum update depth exceeded`가 발생했다. | advisory | 날짜 파생값은 `useMemo`나 문자열/숫자 primitive로 고정하고, effect 의존성에 비안정 객체를 직접 넣지 않는다. |
| Trigger wrapper click mismatch | 커스텀 선택 박스를 `label` 성격 래퍼로 감싸면 실제 클릭 지점과 트리거 영역이 어긋나 펼침 동작이 흔들릴 수 있다. | advisory | 커스텀 select/date trigger는 중립 컨테이너 + 실제 버튼 구조로 유지하고, 클릭 전달을 래퍼에 의존하지 않는다. |
